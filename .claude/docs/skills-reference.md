# Skills Reference (Slash Commands)

All 45 slash commands available in DS Studios, organized by category.

## Onboarding (3)

| Command | Description |
|---------|-------------|
| `/start` | First-time onboarding — detects project state, guides setup |
| `/setup-stack` | Configure technology stack (Python, cloud, warehouse) |
| `/configure-mcp` | Configure MCP server integrations |

## Data (7)

| Command | Description |
|---------|-------------|
| `/ingest` | Set up data ingestion from databases, APIs, files |
| `/eda` | Exploratory data analysis with profiling and visualization |
| `/data-profile` | Quick statistical profile of a dataset |
| `/data-quality` | Comprehensive data quality assessment |
| `/data-catalog` | Document datasets with metadata |
| `/schema-design` | Design data warehouse schemas |
| `/data-lineage` | Trace data from source to consumption |

## Feature Engineering (3)

| Command | Description |
|---------|-------------|
| `/feature-engineer` | Create, validate, and document features |
| `/feature-store` | Set up feature store for training and serving |
| `/feature-select` | Statistical feature selection methods |

## Modeling (7)

| Command | Description |
|---------|-------------|
| `/train` | Model training with experiment tracking |
| `/evaluate` | Comprehensive model evaluation and model cards |
| `/hyperopt` | Hyperparameter optimization with Optuna |
| `/experiment` | MLflow experiment setup and management |
| `/compare-models` | Compare multiple models from experiments |
| `/explain` | Model explainability (SHAP, LIME, PDP) |
| `/automl` | Automated ML for quick baselines |
