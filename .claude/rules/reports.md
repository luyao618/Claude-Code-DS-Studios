# Reporting Rules

- Every report MUST have: title, author, date, purpose, and executive summary
- Methodology section MUST document: data sources, analysis approach, assumptions
- All numbers MUST have context: comparison to baseline, trend, or target
- Visualizations MUST follow the visualization rules (titles, labels, accessibility)
- Recommendations MUST be actionable with expected impact quantified
- Caveats and limitations MUST be explicitly stated
- Data freshness: state as-of date for all data used
- Reports MUST be reviewed by analytics-lead before sharing externally
- Use consistent formatting across all reports
