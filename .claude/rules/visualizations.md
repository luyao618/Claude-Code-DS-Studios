# Visualization Rules

- Every chart MUST have a title, axis labels, and legend (where applicable)
- Use colorblind-friendly palettes (avoid red-green only distinctions)
- Never truncate axes to exaggerate differences
- Include data source and date in chart annotations or captions
- Use appropriate chart types: bar for comparison, line for trends, scatter for relationships
- No pie charts with more than 5 categories
- No 3D charts — they distort perception
- Include units in axis labels
- Figures saved to project/reports/figures/ with descriptive filenames
- Dashboard KPIs MUST show comparison to previous period or target
